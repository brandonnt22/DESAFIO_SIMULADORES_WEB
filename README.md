# Simuladores Matemáticos Interactivos

Proyecto desarrollado para el desafío técnico de programación web.

## Funcionalidades

### 1. Transferencia de Calor
Calcula la temperatura final usando la Ley de Enfriamiento:

T = Ts + (T0 - Ts)e^(-kt)

### 2. Combinaciones de Sorteo
Calcula:

C(n,r) = n! / (r!(n-r)!)

y obtiene el total de combinaciones posibles.
